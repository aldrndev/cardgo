Card Go memprioritaskan privasi Anda. Data finansial Anda disimpan secara lokal (offline) di perangkat Anda.

Kami menggunakan koneksi internet hanya untuk fitur pendukung seperti menampilkan iklan (pada versi gratis) atau verifikasi pembelian fitur premium. Data finansial sensitif Anda TIDAK PERNAH dikirim ke server kami.

ğŸ“ Data yang Disimpan

Semua data berikut disimpan secara lokal di perangkat Anda menggunakan AsyncStorage (React Native):

  • Metadata kartu kredit (nama bank, alias, brand kartu)
  • Informasi keuangan (limit kredit, total tagihan, minimum payment)
  • Siklus tagihan (billing date, due date)
  • 4 digit terakhir kartu (opsional, jika Anda input)
  • Transaksi (tanggal, jumlah, kategori, deskripsi)
  • Langganan berulang (nama, jumlah, tanggal tagihan)
  • Rencana cicilan
  • Riwayat pengajuan kenaikan limit
  • Preferensi notifikasi
  • Catatan pribadi untuk setiap kartu

  • CVV/CVC (3 atau 4 digit di belakang kartu)
  • PIN kartu kredit
  • Tanggal kadaluarsa kartu

Semua data Anda tersimpan hanya di perangkat Anda. Tidak ada data yang dikirim ke server kami atau pihak ketiga manapun.

Jika Anda menghapus aplikasi, semua data akan terhapus permanen dari perangkat Anda.
